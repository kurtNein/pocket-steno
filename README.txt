Thank you for downloading PocketSteno!

This is an application dedicated to transforming .pptx files into text.

The version available on GitHub is the Python script which comprises the process.

A standalone executable (PocketSteno.exe) can be made available for 64-bit Windows 10 via pyinstaller.

To run, clone the script into a .py file, import into your IDE, or double click a PocketSteno.exe file.

If there is a window from your operating system claiming it protected you from running the application:
    > Expand the "More info" box. Then, select "Run anyway".

Follow the instructions displayed on screen.


~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~

To begin, enter a directory from which files will be read.
This program handles an entire folder at a time.
Enter the full path to the folder directory in the form of:

 > 	C:\..\PocketSteno\Presentations


You will have two options regarding how the .pptx files are treated.

You can either:
    > Generate an essay through Open AI. This will also prompt you for an API key provided by Open AI.
    > You can copy and paste any valid key into the field.
OR:
    > Transcribe the contents of the .pptx verbatim. The transcription will be written to a document unchanged.
    > This option does not require a key, nor an internet connection. This is the only option available offline.


Whichever you choose, you will then need to supply a destination folder.

Enter the full path to the folder directory in the form of:

 > 	C:\..\PocketSteno\Results


~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~-~

To ensure the process runs properly, close all files in directories to be used.

In other words, do not have any of the .pptx files open before running the program.
